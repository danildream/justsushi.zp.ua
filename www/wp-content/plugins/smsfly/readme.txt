Отправка смс уведомлений с помощью сервиса sms-fly.com

== Description ==

**Данный плагин отправляет смс уведомления на телефон администратора или клиента о событиях на сайте, в том числе отслеживание заказов в плагине WooCommerce, через сервис [sms-fly.com](https://sms-fly.com)**

**Возможности:**

Уведомления на телефон по смс о событиях на сайте.

* Уведомление о публикации нового поста.
* Уведомление о обновлении поста.
* Уведомление о том что пользователь залогинился на сайт.
* Уведомление о установке нового плагина.
* Уведомление о обновлении плагина.
* Уведомление о установке новой темы.
* Уведомление о обновлении темы.

Уведомления на телефон по смс о заказах плагина WooCommerce.
* Уведомление о новом заказе на телефон администратора.
* Уведомление о новом заказе на телефон клиента.
* Уведомление о смене статуса заказа на телефон администратора.
* Уведомление о смене статуса заказа на телефон клиента.

== Installation ==

1. Upload `smsfly` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
